# Tiny Bubbles — Prioritized Feature Matrix

*Compiled 2026-06-27. Companion to `00-SYNTHESIS.md`. Each feature is justified by (a) market/community demand and (b) verified science (using the adversarial fact-check, not the original overstated claims), and mapped to the MIT source repo + specific files to lift, with a build-effort estimate.*

**Tiers:** **MVP** = ship in v1.0, the loop doesn't work without it · **v1** = fast follow, 1–2 releases after launch · **later** = differentiator/expansion once retention is proven.

**Effort scale:** Low (re-theme/copy edits) · Med (decouple + adapt + reskin) · High (substantial new work).

**Source paths** are relative to `/Users/ameyakashid/Desktop/adhd india/_sources/`. "In-stack" = already RN/Expo TS. "Port" = different stack, lift the logic and reimplement in RN.

## Cross-cutting build notes

- **Assembly order (from `50` §4):** base = `lockin` → graft `habit-tracker-app` component kit + pickers + charts + settings → port `tether` gamification engine (AsyncStorage-backed) → port `momentum` streak math + mood + "add a step" → optionally port `sidejot` chunking prompt (AI off by default).
- **Mandatory pre-ship rework on `lockin`:** reverse the shame/"mockery" tone (angry sprites, "ruthless advisor" AI), replace the rigid one-immutable-goal-per-year model with flexible short-horizon tasks, redraw art + brighten palette, align SDKs (habit-tracker-app SDK 50 vs lockin SDK 54 — lift components not the shell), keep AI off by default, drop all surveillance pipelines.
- **License hard rule:** `adhd-india` (no license) and `adhd-focus-mate` (MIT in README, no LICENSE file) are **reference-only — do not ship their code.** All other recommended donors are verified MIT; attribute Apache-2.0 transitive libs.
- **The throughline (from `40` §12):** a forgiving, immediate, low-friction token economy around a beloved, never-punishing companion, on a calm canvas with celebratory moments, with curated autonomy, everything spoken aloud for non-readers — and a deliberate refusal of the engagement-maximizing dark patterns the same toolkit makes easy.
